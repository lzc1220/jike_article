## 安装与运行

项目运行之前，请确保系统已经安装以下应用

- node (6.0 及以上版本)

### 运行命令

```
git clone git clone git@github.com:lzc1220/jike_article.git
cd jike_article/react-jike

npm install

#development
npm run start

#production
npm run build
```
